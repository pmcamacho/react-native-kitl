# Install packages and run app

From root folder splash-screen-app:

ios and android:
```
npm install
```

ios: 

```
cd ios
pod install
cd ..
```

android and ios:

```
npm run start
```

android: 

```
npm run android
```

ios: 

```
npm run ios
```
